A little personal project:
The intention was that I've always read a lot of philosophy and it's entered my life via osmosis, (podcasts and books) but I'd really like to formalise it and make it a part of every day.

It's intended as a little daily routine reminder ive added the ability to scrub my philosophy and replace it with your own mantra. 
Data is saved locally so noone can see your answers, but that means if you change browsers its like starting from scratch.

So since the data could potentially be lost i added the ability to print or save your history, which makes it more like the intended "journalling" I was after!
